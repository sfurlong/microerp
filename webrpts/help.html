What are restrictions?

Once you have chosen certain columns from a table you can restrict your query so that only certain rows in those columns are returned. You can specify a value and get only those rows that are equal/not equal to that value, or greater/less than it, etc. These criteria are added to the "where" clause in the SQL statement.

Specifying a sort column

If you want your results to be sorted somehow, you can specify that here. Simply choose which column you want as the sort criteria and whether you want the results in ascending (1-9, A-Z) or descending (9-1, Z-A) order. This gets placed in the "order by" clause of a SQL statement, which sorts in ascending order by default.

Saving a custom report

This feature allows you to save a custom report for future use. Simply enter a title and the SQL statement you used to create the report. The next time you want to run the report you can just click on its link instead of generating the SQL again.

Viewing a Report in Excel Format

You can choose to view the results of a report in Microsoft Excel format. When you check the box marked "View in MS-Excel format" your report will be displayed in an Excel window, either embedded in your browser, or as a separate window. From here you will have the option to print or save the file just as if it were a standard .XLS file.

If after you click submit you see a file download dialog box, select "Open file from its current location" and click OK.
