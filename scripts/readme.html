microERP Install QuickStart

  1. By default the install is configured to point to an embedded demo database using HSQLDB (Hypersonic SQL Database) and an embedded http server (Jetty).
  2. After the Install go to windows Start Menu / Program Files and find the microERP program group.
  3. Choose the "StartServer" program option. This will start the embedded database server and the HTTP server.
  4. Next choose the "microERP Client" program option to start the microERP user interface.
  5. The userid and password for the embedded demo database are:  UserId=”demo”  Password=”demo”
